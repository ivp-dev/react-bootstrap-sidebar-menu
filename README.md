Description

Installation
------------------------------------------------------------------------------

```

npm install react-bootstrap-sidebar-menu

```

Usage
------------------------------------------------------------------------------

```

import SidebarMenu from 'react-bootstrap-sidebar-menu';

```


Compatibility
------------------------------------------------------------------------------

* React v16.14.0 or above,
* React-Dom: v16.14.0 or above
